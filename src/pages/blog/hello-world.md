This is an example blog post. You can write in plain **Markdown** and it'll be styled automatically.

## How to add a new post

Just create a new `.md` file in `src/pages/blog/` with frontmatter like this:

```markdown
---
layout: ../../layouts/Blog.astro
title: "Your Post Title"
date: "2026-04-10"
description: "A short description for SEO."
---

Your content here.
```

That's it — Astro handles the rest.
